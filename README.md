# 2GIS_test_assignment

## Описание
Небольшой Python-проект для тестирования публичных **API 2GIS**:
- Routing API — построение маршрута на автомобиле между двумя точками в Москве.
- Search API — поиск организаций поблизости и работа с их атрибутами.
Проект демонстрирует навыки:
- работы с REST API и анализом JSON-ответов;
- написания автотестов на **pytest**;
- построения отчётов с **Allure**;
- реализации позитивных и негативных сценариев;
- интеграционного теста, объединяющего поиск и построение маршрута.


## Установка и настройка

1. Клонируйте репозиторий:

git clone https://github.com/leondc13/2GIS_test_assignment.git

2. Установите зависимости:
pip install -r requirements.txt

3. Добавьте свой API-ключ в settings.py

## Запуск тестов (+ отчет allure):
pytest -v --alluredir=allure-results && allure serve allure-results

## Набор тестов

test_routing.py	
- Позитивное построение маршрута
- Ошибочные координаты
- Отсутствие API-ключа

test_search.py	
- Поиск кафе в Москве
- Поиск с дополнительными полями
- Неверный ключ

test_search_and_route.py	
- Интеграция: поиск ближайшей АЗС → построение маршрута к ней
